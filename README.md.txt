# Social Media Analytics Tool

## Description
Social Media Analytics Tool est une application Python qui permet d’analyser les performances des posts sur les réseaux sociaux. Elle fournit un tableau de bord interactif pour visualiser le nombre total de posts, likes, commentaires, partages, top posts par engagement et hashtags les plus populaires.
